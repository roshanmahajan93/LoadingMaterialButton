# LoadingMaterialButton

## Usage

Minimum SDK: 21 

Currently this widget is running with androidX and also allows the usage of databinding to set the elements comfortably.

### Gradle
```
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}
```
```
dependencies {
	implementation 'com.github.roshanmahajan93:LoadingMaterialButton:1.0.0'
 }
```
